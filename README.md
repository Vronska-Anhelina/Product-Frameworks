# Product-Frameworks

A collection of practical tools and templates for product analytics.

## Cohort Analysis Framework

A structured framework for running cohort analysis on user or customer data.

### What's included

- Python scripts for cohort retention calculations
- Visualization templates (heatmaps, retention curves)

### Use cases

- Retention analysis by signup cohort
- Revenue retention (net/gross)
- Feature adoption tracking over time
- Churn pattern identification


### Requirements

- Python 3.9+
- pandas, matplotlib, seaborn
